Project: AI Website Copy Generator for Local Businesses
Objective

This project demonstrates a structured prompt engineering system that generates conversion-focused website copy for local businesses.

Prompt Architecture

The system includes:

Role definition (conversion copywriter + UX strategist)

Input variable structure

Defined output sections

Constraint-based quality control

Anti-cliché filtering

Tone adaptation

Local relevance integration

Test Case

Local Gym –  Professional tone

Outputs

Generated homepage copy, services section, CTAs, and FAQs using the structured master prompt.
